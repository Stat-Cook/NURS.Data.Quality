
# Overview

Implementation of data quality (DQ) techniques as used in the NuRS study
into staff retention and patient safety. The package implements a series
of DQ tools for standalone analysis and automated report generation.

``` r
# Install from github via:

install.packages("devtools")
devtools::install_github("Stat-Cook/NURS.Data.Quality")

# Or install a development branch from github via:
devtools::install_github("Stat-Cook/NURS.Data.Quality", ref="BRANCH_NAME")
# e.g.
devtools::install_github("Stat-Cook/NURS.Data.Quality", ref="boolean_flip")
```
